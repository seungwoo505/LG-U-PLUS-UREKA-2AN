배열은 불변이다.

-----------------------------

extened - 상속은 기본적으로 하나

implements - 만약 상속이 더 필요하다면 다중 상속 가능

--------------------------------------------------

extened와 implements의 차이점

extened - 부모에서 선언 / 정의를 모두 구현 -> 자식은 메소드 / 변수를 사용하거나 오버라이딩을 하여 사용할 수 있다.

implements - 부모는 선언만 하며 정의는 자식에서 오버라이딩을 하여 사용할 수 있다.
